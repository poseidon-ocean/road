# 漫画动画

## 污泥能令莲花更光华， 磨砺能使钻石更闪耀
```
｜贤二如是说：所有遇见都是美好｜

贤二跟菜鸟说，王小五又被
警察叔叔叫去喝茶了。
菜鸟说，这跟他一直以来的
暴躁情绪有关。
贤二说，赵小翠去看心理医生，
花了好多钱。
菜鸟说，这跟她一直以来的
负面情绪有关。
贤二说，吴美丽又失恋了。
菜鸟说，这跟她一直以来
内心里排斥现实有关。
贤二说，钱老板又要打官司了。
菜鸟说，这跟他在一直以来
浮躁的情绪有关。
贤二说，张小四又失业了。
菜鸟说，这跟他一直以来
喜欢抱怨有关。

贤二说，听说菜鸟家族知道一些
非常不吉祥的咒语，
会让人倒霉，你能告诉我吗？
菜鸟说，是的，“我的命真苦”
“为什么要我做这些事情”这样的
负面情绪的话一定不能说出口，
说出来，好运气就没了。
贤二说，太好了，太有道理了。
可是……
菜鸟说，可是什么？
贤二说，我刚跟师父说完这两句话……

浮躁、暴躁，
是因为自己一直在排斥
眼前的因缘，没有安住当下。
内心一直在串习“我真命苦”
“为什么要我做这些事情”等负面情绪，
只会越来越深地陷入烦恼泥潭。
要转一个心态去面对，
接纳现状，全心成长。
污泥能令莲花更光华，
磨砺能使钻石更闪耀。
```
